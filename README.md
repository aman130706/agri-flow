# AGRII FLOW 🌱

AGRII FLOW is a modern web application designed to streamline agricultural workflows and data management. Built with a fast, scalable frontend stack, it delivers a clean UI, smooth performance, and an excellent developer experience.

---

## 🚀 Tech Stack

- **Vite** – Lightning-fast build tool
- **React** – Component-based UI
- **TypeScript** – Type-safe JavaScript
- **Tailwind CSS** – Utility-first styling
- **shadcn/ui** – Accessible, customizable UI components

---

## 📦 Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v18+ recommended)
- **npm** (comes with Node)

> Tip: Install Node using [nvm](https://github.com/nvm-sh/nvm) for easier version management.

---

### Installation

Clone the repository and install dependencies:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
